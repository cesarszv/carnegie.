# Carnegie

![logo](assets/logo.png)

![banner](assets/banner.png)

**Carnegie** es un CRM personal: un lugar privado y self-hosted para gestionar
las personas de tu vida. Guarda quién es cada persona, qué pasó este año y
cuándo se vieron, todo en un solo lugar que te pertenece.

Tus relaciones, capas sobre el tiempo. Privado por diseño. Tus datos son tuyos.

## Landing

Esta carpeta contiene la landing page pública de Carnegie. Es un sitio
estático: HTML, CSS y JavaScript vanilla, sin frameworks ni build step.
Pensado para servirse directamente desde GitHub Pages.

### Estructura

```
index.html       Página principal
css/style.css    Estilos
js/main.js       Animaciones, menú móvil, formulario
assets/          Logo y banner
.nojekyll        Sirve archivos tal cual en GitHub Pages
```

### Verlo

Abre `index.html` en tu navegador, o visita el sitio publicado:

> [Carnegie Landing](https://cesarszv.github.io/carnegie/)
